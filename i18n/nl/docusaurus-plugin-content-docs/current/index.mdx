# Welkom

Wil je deel uitmaken van de green software beweging?

Als je jouw softwareapplicaties duurzamer wilt maken en een carrière in green software wilt opbouwen, dan is deze training voor jou. Het leert je hoe je groenere applicaties kunt bouwen, onderhouden en draaien, ongeacht het toepassingsdomein, de industrie, de organisatiegrootte of het type, de programmeertaal of het framework.

## Wat is een software practitioner?

Een software practitioner is iedereen die betrokken is bij het bouwen, implementeren of beheren van een softwareapplicatie. Dit omvat onder meer:

- Developers
- Dev Ops
- Testers
- Architects
- SREs
- Product Managers
- UX Designers

## Hoe haal je het beste uit deze cursus?

- Heb ten minste **één jaar werkervaring** met het bouwen, implementeren of beheren van softwareapplicaties.
- Doorloop elke sectie op volgorde van begin tot eind.
- Maak de kennischecks aan het einde van elke sectie.

## Aan het einde van deze training begrijp je het volgende:

- De belangrijkste drijfveren van carbon emissions voor softwareapplicaties.
- De 3 acties die je kunt ondernemen om de carbon emissions van jouw software te verminderen.
- Hoe je software emissions kunt meten met behulp van het Greenhouse Gas (GHG) protocol of de Software Carbon Intensity (SCI) specificatie.
- De verschillende soorten bedrijfsklimaatdoelen, zodat je jouw green software voorstellen kunt afstemmen om de publieke verplichtingen van jouw organisatie te helpen nakomen.

## Wie heeft deze training gemaakt?

Deze training is gemaakt door het Principles & Patterns team in de [Community Working Group](https://community.greensoftware.foundation) van de [Green Software Foundation](https://greensoftware.foundation)

De training is open source en creative commons, de broncode voor deze site is te vinden op onze GitHub https://github.com/Green-Software-Foundation/training.
