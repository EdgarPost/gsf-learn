Voulez-vous faire partie du mouvement du logiciel durable?

Si vous voulez rendre vos applications logicielles plus durables et établir une carrière dans les logiciels durables, alors cette formation est pour vous. Elle vous apprendra à créer, maintenir et exécuter des applications plus écologiques, quel que soit le domaine d’application, l’industrie, la taille ou le type d’organisation, le langage de programmation ou le système.

Si vous réussissez l’examen [associé](https://grnsft.org/practitioner/lf-exam), il est accompagné d’un certificat d’achèvement de la foundation Linux. Certains employeurs ont créé des formulaires pour faciliter la gestion des crédits pour avoir complété ce cours. Si votre employeur est [dans cette liste](https://docs.google.com/document/d/1Jv-f7wGyODzEySQFfVk6Vo_OAHWjf-jAdFmyfDQUgvU/edit?usp=sharing), veuillez cliquer sur le nom de votre organisation pour enregistrer vos crédits. Si votre organisation souhaite ajouter son formulaire à la liste, veuillez contacter notre équipe à help@greensoftware.foundation.

## Qu’est-ce qu’un professionnel du logiciel?

Un professionnel du logiciel est toute personne impliquée dans la pratique du dévelopement, du déploiement ou de la gestion d’une application logicielle. Cela comprend, sans s’y limiter :

- Les developpeurs
- Les Dev Ops
- Les testeurs
- Les architectes
- Les SREs
- Les responsables de produit
- Les designers UX

## Comment tirer le meilleur parti de ce cours?

- Avoir au moins **un an d’expérience dans le secteur** de la création, du déploiement ou de la gestion d’applications logicielles.
- Remplir chaque section dans l’ordre du début à la fin.
- Effectuer les vérifications des connaissances à la fin de chaque section.
- Si vous souhaitez obtenir un certificat d’achèvement, passez l’examen via la foundation Linux.

## À la fin de cette formation, vous aurez acquis les connaissances suivantes:

- Les principaux moteurs des émissions de carbone pour les applications logicielles.
- Les 3 mesures que vous pouvez prendre pour réduire les émissions de carbone de votre logiciel.
- Comment mesurer les émissions des logiciels à l’aide du protocole Gaz à Effet de Serre (GES) ou de la spécification Intensité Carbone du Logiciel (SCI - Software Carbon Intensity).
- Les différents types d’objectifs climatiques de l’entreprise afin que vous puissiez aligner vos propositions de logiciel durable pour aider à respecter les engagements publics de votre organisation.

## Qui a créé cette formation?

Cette formation est créée par l’équipe Principes et Modèles du [groupe de travail communautaire](https://community.greensoftware.foundation) de la [Green Software Foundation](https://greensoftware.foundation).

La formation est open source avec une license creative commons, les sources de ce site peut être trouvée sur notre GitHub https://github.com/Green-Software-Foundation/training.
