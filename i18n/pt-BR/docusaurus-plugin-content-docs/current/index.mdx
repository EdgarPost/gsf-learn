Você deseja fazer parte do movimento de software verde?

Se você deseja criar aplicações mais sustentáveis e construir uma carreira em software verde, então este treinamento é para você. Você irá aprender como construir, manter e rodar aplicações mais verdes indpenendente do domínio da aplicação, indústria, tamanho da organização ou tipo, linguagem de programa ou estrutura.

Se você for aprovado(a) no exame [associated exam](https://grnsft.org/practitioner/lf-exam), você receberá uma certificação da Linux Foundation. Alguns empregadores criaram um formulário para facilitar a administração de créditos por completar esse curso. Se seu empregador estiver [nesta lista](https://docs.google.com/document/d/1Jv-f7wGyODzEySQFfVk6Vo_OAHWjf-jAdFmyfDQUgvU/edit?usp=sharing), por favor, clique no nome da sua organização para registrar os créditos. Caso sua organização tenha interesse em participar e adicionar seu nome à lista, entre em contato através do endereço eletrônico help@greensoftware.foundation.

## O que é um profissional de software?

Um profissional de software é qualquer pessoa que está envolvida na prática de construir, entregar ou gerenciar aplicações computacionais. Estes podem ser (mas não exclusivamente):

- Desenvolvedores
- Dev Ops
- Testadores
- Arquitetos
- SREs
- Gerentes de Produto
- UX Designers

## Como tirar o máximo proveito do curso?

- Tenha pelo menos **um ano de experiência** construindo, entregado ou gerenciando aplicativos.
- Complete cada seção na devida ordem, do início ao fim.
- Complete os testes de conhecimento ao final de cada seção.
- Se quiser ganhar um certificado, preste o exame através da Linux Foundation.

## Ao final deste treinamento, você será capaz de entender os seguintes tópicos:

- Os principais impulsionadores das emissões de carbono para aplicações de software.
- 3 ações que você pode tomar para reduzir emissões de gás carbônico com seus aplicativos.
- Como medir emissões de aplicativos usando o protocolo de Gás de Efeito Estufa ou a especificação de Intesidade de Carbono do Software.
- Os diferentes tipos de alvos climáticos das corporações de modo que você seja capaz de alinhar suas propostas de software verde e alcançar os compromissos públicos de sua organização.
